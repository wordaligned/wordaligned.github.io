I used the PythonWin IDE (available with the Active Python distribution) while developing this script, and can recommend it to Windows users. The context sensitive help is great, the debugger is good, and the key-bindings – incredibly – include my favourites from both GNU emacs and Microsoft Visual Studio.

My thanks to Dan Tallis for reviewing an earlier draft of this article, and for inspiring me to learn Python.
