Saving changes to read-only files

2006-08-01 • • Comments

So I click on a file and it warns me someone else has the file locked for editing. Do I wish to open a temporary copy, and merge my changes in later? Or open a read-only copy? Read-only, please, I reply, as always (I don’t trust Microsoft Word’s merge, why can’t it remember that?) I read the document. I don’t change it. I try to close it. It asks me if I want to save my changes!?
